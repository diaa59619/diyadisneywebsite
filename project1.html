📊 Growth Analytics Engine

Overview

This project focuses on analyzing user growth, retention, and drop-off across the user journey using structured data analysis techniques.

Problem

Users were dropping off at different stages of the journey, impacting overall retention and growth.

Skills Used

SQL Python Excel Tableau Data Cleaning Funnel Analysis Cohort Analysis

Workflow

Data Collection → Data Cleaning → Funnel Analysis → Cohort Analysis → Behavioral Analysis → Dashboard Creation → Insight Communication

Key Insights

  • Major drop-off during onboarding stage
  • Low engagement after initial interaction
  • Retention significantly drops after Day 2

Business Relevance

This project helps businesses identify critical drop-off points in the user journey and optimize onboarding and engagement strategies to improve retention and long-term growth.

Outcome

Proposed onboarding improvements and engagement strategies that can increase retention and improve user lifetime value.
